## ¿Para qué sirve grabar en la nube?

Este tipo de almacenamiento es cada vez más solicitado por entes reguladores para garantizar que todo marche en buenas condiciones, por lo que grabar en la nube un DVR o cámara IP es algo primordial.

¿Es siempre necesario un servicio Cloud externo, o una empresa puede tener su propio datacenter? Para responder a esta pregunta, se deben tener en cuenta dos aspectos, el práctico y el económico. Primero y principal, las cámaras necesitan de un circuito para funcionar. Deben instalarse, configurarse, tender un cableado técnico específico, implementar horas de trabajo manual para que todo quede en perfectas condiciones. Y dicho trabajo requiere de una inversión sustancial.

Ahora bien, si a todo esto agregamos además la creación de un centro dedicado al almacenamiento, que requiere su propio servicio de mantenimiento, cuidado, reacondicionamiento y actualización, nos damos cuenta que estamos dejando de lado los dos principales aspectos antes mencionados.
