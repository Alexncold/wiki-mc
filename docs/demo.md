## Requisitos del Navegador

Para una experiencia óptima, te recomendamos usar:

- Google Chrome o Microsoft Edge (últimas versiones)
- Conexión a Internet estable
- Resolución mínima recomendada: 1280x720

:::note
Si experimentás problemas al cargar la demo, intentá:
1. Limpiar la caché del navegador
2. Deshabilitar extensiones que puedan interferir
3. Probar en una ventana de incógnito
:::
